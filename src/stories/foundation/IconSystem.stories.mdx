# Icon System

## Introduction

The icon system of this project is separated into multiple **SVG Sprites**, which one with a generic category. The list below show you all the icons present on each category.

## Categories

{Object.entries(icons).map(([category, icons]) => (

<section>
    <h3><strong className={'capitalize'}>{SpriteCategories[category]}</strong> icon(s)</h3>

    {!icons.length && <p>No icons on this category!</p>}

    <IconGallery>
        {(icons ?? []).sort().map(icon => (
            <IconItem key={`${category}-${icon}`} name={icon}>
                <Icon className={'text-zinc-900'}
                      category={SpriteCategories[category]}
                      name={icon}/>
            </IconItem>
        ))}
    </IconGallery>

</section>
))}
